# Rate Limiter App

A Rate Limiter project using FastApi, Pydantic, Pytest, Poetry and Docker.

## Run locally
```
docker-compose up --build
```


## Run Test
```
docker-compose run api pytest
```

## To build and run:
```
docker-compose up --build
```

## Check docs
1. Swagger
```
http://localhost:8000/docs
```

1. Redocly
```
http://localhost:8000/redoc
```
